If you type in blacksheepwall you will see all of the AI's Pieces. 

Type stats in the console to display players overall score.

To start a game click new game and then place your pieces. Once the last piece is placed you will be able to click Start Game. Doing so will place all the AI pieces and you will now make the first move.

Forfeit will end the game and allow you to start a new one.

When you close the application all the account data is saved to a text file and when you start the application. 

When you open the application, the app will make a new textfile if never made and if it was then it reads in the serialized data.
